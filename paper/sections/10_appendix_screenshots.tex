\section{Application Screenshots}
\label{app:screenshots}

This appendix shows screenshots of two interfaces released alongside the
corpus: the human-review web application
(\texttt{src/review\_app/}, port 5556) used to collect WSET-Diploma
gold-sheet ratings, and the project monitoring dashboard
(\texttt{src/dashboard/}, port 5555) used during construction.

\subsection{Human-review web application}

The review app is a Flask application with two layers of authentication
(outer HTTP Basic for shared-link access, inner per-reviewer session
cookies) and writes ratings into a separate \texttt{human\_reviews}
Postgres table to keep gold-sheet labels independent of automated
audit findings. Each reviewer is associated with a WSET certification
level recorded at registration; the app supports multi-reviewer IRR
($\kappa$) computation out of the box.
